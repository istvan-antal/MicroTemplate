MicroTemplate::assign($name,$value) - Assigns $value to $name in the template scope.

MicroTemplate::display($template) - Constructs the output based on $template.php located in the template directory.

MicroTemplate::fetch($template) - Similar to MicroTemplate::display($template) but returns the output.

MicroTemplate::setTemplateDirectory($dir) - Can be used to configure the directory where all templates reside in.
